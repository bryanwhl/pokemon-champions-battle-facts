# Pokemon Champions Battle Facts

Agent Skill for grounding Pokemon Champions and VGC research in verified battle facts.

The skill includes deterministic type matchup tooling, PokeAPI lookup tooling, a Smogon damage-calculator wrapper, and source-policy references that force agents to distinguish stable mechanics from current Pokemon Champions legality and metagame claims.

Use it by installing or importing this repository as an Agent Skill. The skill root is this repository root and the required entrypoint is `SKILL.md`.
